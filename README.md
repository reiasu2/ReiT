# ReiT

ReiT is Rei's personal Codex workflow layer. It keeps the useful engineering discipline from Superpowers, but routes tasks through a lighter self-maintained skill set.

## Skills

- `ReiT`: main router, task sizing, safety, verification levels, and output rules.
- `ReiT-brainstorming`: lightweight requirements and option exploration, with full mode for complex design.
- `ReiT-plan`: plan-first workflow for medium, large, ambiguous, multi-step, or multi-file tasks.
- `ReiT-orchestrator`: controller-brain mode for complex or multi-task work, primarily saving main-context space through trigger scoring, task packets, compact ledgers, and bounded subagent delegation.
- `ReiT-debugging`: root-cause-first debugging.
- `ReiT-verification`: completion and delivery verification gate.
- `ReiT-review`: local review and review feedback handling.
- `ReiT-worktree`: branch/worktree hygiene and closeout.

## Install

Copy the folders under `skills/` into:

```text
%USERPROFILE%\.codex\skills
```

Then copy `AGENTS.md` to:

```text
%USERPROFILE%\.codex\AGENTS.md
```

Restart Codex after installing or updating skills.
